Pri pokretanju web sajta potrebno je:
-Instalirati xampp jer sam koristio php ekstenziju(u php fajlovima je html kod)
-Pokrenuti xampp program i startovati apache i mysql
-Fajl projekat BildStudioProject treba kopirati na lokaciju C:\xampp\htdocs\(C:\xampp\htdocs\BildStudioProject)
-Pri pokretanju projekta biti povezan na internet(zbog korišæenih linkova)
-Otvoriti browser i u polje za pretragu kucati localhost/BildStudioProject
